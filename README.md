## Overview
This repository contains an interactive Tableau dashboard that analyzes sales and performance metrics with a focus on actionable insights. It includes the packaged workbook, sample dataset, annotated screenshots, and an  GitHub Pages embed.

### Key highlights
- Clean, filter-enabled visualizations for fast decision-making. 
- KPI summary, trend analysis, and category breakdowns. 
- Designed for non-technical stakeholders and business teams.


## How to view
1. View the interactive dashboard on Tableau Public (link above).  
2. Or download *Tableau Reader* (free) / use Tableau Desktop and open workbook/Book2.twbx.

## Files included
- workbook/Book2.twbx - Packaged workbook (or .twb if data separated)  
- data/sample_data.csv - Small anonymized sample dataset and schema
